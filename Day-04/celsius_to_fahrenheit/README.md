# Celsius to Fahrenheit Conversion

## Problem
Convert temperature from Celsius to Fahrenheit.

## Formula

Fahrenheit = (Celsius × 9/5) + 32

## Solution

```python
celsius = float(input("Enter temperature in Celsius: "))

fahrenheit = (celsius * 9/5) + 32

print(fahrenheit)
