# Access List Using a for Loop

## Problem

Access and print each element of a list using a `for` loop.

## Solution

```python
fruits = ["Apple", "Banana", "Mango", "Orange", "Grapes"]

for fruit in fruits:
    print(fruit)
```

## Sample Output

```text
Apple
Banana
Mango
Orange
Grapes
```
